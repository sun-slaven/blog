git操作的一些记录

2016-03-26

在cd到某一个目录下时,往往是git clone “URL”,
也可以是

  1. git init(创建空白.git文件即空白代码库)
  2. git remote add origin “URL”(此时.git/config 文件会记录下远程代码库,该文件可查看)
  3. 在第一次push时,git push -u origin “分支名”
